# Multi-Dimensional-Academic-Intelligence-System
This project is a Python-based academic analytics system designed to analyze student performance using multiple parameters such as marks, attendance, and assignment scores. The system simulates student data and performs statistical analysis to generate insights about academic performance.

Description
-----------
This Python project implements a Multi-Dimensional Academic Intelligence System to analyze student performance using different academic parameters such as marks, attendance percentage, and assignment scores.
The system generates student data and performs statistical analysis to evaluate academic performance and identify patterns. The project uses Python libraries like Pandas and NumPy along with basic data structures such as lists, tuples, and dictionaries to organize and process the data efficiently.

How It Works
------------
The program first takes the roll number from the user and uses the last digit to determine the number of students to generate. Random values are generated for marks (0–100), attendance (0–100), and assignment scores (0–50).
The student data is stored using lists and tuples and then converted into a Pandas DataFrame for analysis. The system classifies students into different categories such as At Risk, Average, Good, and Top Performer based on their marks and attendance.
The program also calculates statistical measures including mean, median, and standard deviation of marks, and determines the correlation between marks and attendance. A performance index is calculated using a formula that combines marks, assignment scores, and attendance to evaluate overall performance.
Finally, the system detects patterns such as academic consistency, attendance risk, and high achievement, and provides a final academic insight

Purpose
-------
The purpose of this project is to demonstrate the use of Python data structures, functions, and libraries like NumPy and Pandas for performing academic data analysis. It helps in understanding how data can be processed and analyzed to generate meaningful insights about student performance.
